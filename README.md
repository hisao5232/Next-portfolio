# Go into-PG-World

Connecting Repair Engineering and Software Development.

## 概要

建設機械の修理エンジニアとしての現場経験と、青山学院大学理工学部で培った理論的思考を融合させたポートフォリオサイトです。
Pythonによる業務自動化から、Next.jsを用いたモダンなWebフロントエンド開発への挑戦を記録しています。

## 技術スタック

- Framework: Next.js 15 (App Router)

- Styling: Tailwind CSS v4

- Language: TypeScript / JavaScript

- Deployment: Cloudflare Pages

## プロジェクトの背景

1983年生まれ。長年、重機のメンテナンスという「物理的な課題解決」に従事してきました。
現在は、その論理的なアプローチをプログラミングの世界に応用し、現場のDX（デジタルトランスフォーメーション）やITによる効率化を目指しています。

## 開発環境の実行方法

ローカル環境で開発サーバーを起動するには以下のコマンドを実行してください。
```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開くと確認できます。
